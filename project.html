PROJECTS HIGHLIGHTS

Eater and Passer Game:

"This project is a simple game application built using C programming fundamentals. The game involves a player who must navigate through a grid, eating food pellets while avoiding obstacles. The player's score increases with each food pellet consumed. The game demonstrates basic C programming concepts such as variables, data types, loops, and conditional statements".

University Management System:

"This project is a comprehensive university management system built using Java programming and NetBeans. It provides a user-friendly interface for administrators to manage student records, courses, faculty information, and exam schedules. The system utilizes object-oriented programming concepts such as classes, objects, inheritance, and polymorphism to organize and manage data efficiently".

Medical Management System:

"This project is a medical management system built using Java programming and data structures. It allows healthcare professionals to manage patient records, prescriptions, and appointment schedules. The system employs data structures such as arrays, linked lists, and trees to efficiently store and retrieve medical data. It demonstrates the application of data structures in real-world scenarios".

Pepsi Advertisement:

"I utilized Adobe Photoshop to create a visually appealing advertisement for Pepsi. By leveraging various tools and techniques, I designed a compelling image that effectively communicates the brand's message. The advertisement features a creative combination of textures, colors, and graphics, showcasing my proficiency in image manipulation and editing".
